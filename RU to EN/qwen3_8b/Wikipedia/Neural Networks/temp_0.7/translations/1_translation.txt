Choosing data for network training and their processing is the most complex stage of solving the task. The training data set must meet several criteria:

Original data is transformed into a form that can be fed into the network inputs. Each record in the data file is called a training pair or training vector. A training vector contains one value per network input and, depending on the type of training (supervised or unsupervised), one value per network output. Training the network on raw data typically does not yield high-quality results. There are several ways to improve the network's perception.
