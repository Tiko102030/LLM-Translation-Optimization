We should consider some of these stages in more detail. Data collection for training

Choosing data for network training and their processing is the most complex stage of solving the task. The training dataset should meet several criteria:

Original data is transformed into a form that can be provided to the network inputs. Each record in the data file is called a training pair or training vector. A training vector contains one value per network input and, depending on the type of training (supervised or unsupervised), one value per network output. Training the network on a raw dataset usually does not yield high-quality results. There are several ways to improve the network's ability to process data.
