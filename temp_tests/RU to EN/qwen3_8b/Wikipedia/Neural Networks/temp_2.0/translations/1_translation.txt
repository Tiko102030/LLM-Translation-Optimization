Data collection for training Choosing data for training the network and its processing is the most challenging stage of solving the task. The training data set must satisfy several criteria: Original data is transformed into a format in which it can be provided to the network inputs. Each record in the data file is called a training pair or a training vector. A training vector contains one value for each input to the network and, depending on the type of training (with teacher or without), one value for each output of the network. Training the network on raw data usually does not yield satisfactory results. There are several ways to improve the network's "perception."
