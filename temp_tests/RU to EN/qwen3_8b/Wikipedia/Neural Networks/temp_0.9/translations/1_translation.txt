We should consider some of these stages in more detail. Data collection for training

Selecting data for network training and processing it is the most complex stage of solving the problem. The training dataset must meet several criteria:

Original data is transformed into a form that can be fed into the network inputs. Each record in the data file is called a training pair or a training vector. A training vector contains one value per network input and, depending on the type of learning (supervised or unsupervised), one value per network output. Training the network on raw data typically does not yield high-quality results. There are several methods to improve the network's perception.
